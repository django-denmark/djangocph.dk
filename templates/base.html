Django Saturday - Open data edition - Hackathon #5

Saturday, March 28, 2015

This time we have joined forces with the good people from Virk Data.

Together with Virk.dk (Erhvervsstyrelsen) we will host a hackathon with the focus on open governmental data.

For a long time we as developers have wanted more open public data. Now we actually got it.

Virk Data currently lists 178 datasets and there are many other sources such as http://data.kk.dk/ and http://www.odaa.dk/

On the day we will bring historical CVR (central company register) data in a Django friendly form.

If you're new to Open Data, then we recommend taking a look at some of the following resources:

  • Virkdata Open Data School
  • Analysis of Danish financial statements example
  • Python: Getting Started with Data Analysis
  • Webjournalistens værktøjskasse - link collection
  • Inspiring examples of how the data can be used

Sign up here: http://www.meetup.com/The-Copenhagen-Django-Meetup-Group/events/220508243/
